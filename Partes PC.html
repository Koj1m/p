Partes internas de un computador





Aquí se enlistan las partes más importantes para el funcionamiento de un computador.




- Placa base/tarjeta madre:

Es una parte fundamental para armar cualquier computadora. En ella se instalan cosas como el microprocesador, la memoria RAM, las trjetas de expansión, etc.

- Microprocesador:

El microprocesador (abreviado CPU) es el encargado de ejecutar todas las tareas del computador, las cuáles son proyectadas a través del sistema operativo.

- Disco duro:

El disco duro almacena la información guardada en el computador para poder ser utilizada en cualquier otro momento.

- Ventilador:

Se encarga de refrigerar todas las partes internas para así evitar que se recalienten y presenten problemas, o incluso, se puedan dañar.

- Memoria RAM:

Es un sistema de almacenamiento, sin embargo, a diferencia del disco duro, sólo almacena la información temporalmente. En ella se cargan y procesan las órdenes dadas al microprocesador.

- Procesador gráfico:

(Abreviado GPU) es una unidad que trabaja cooperativamente con el microprocesador para aligerar su trabajo, se encarga, como su nombre lo indica, de procesar los gráficos que se presentn en la pantalla, y tiene una unidad de RAM exclusiva para ella, llamada VRAM (Memoria RAM Virtual).
